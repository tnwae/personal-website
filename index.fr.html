    Langues de programmation

    Pour lire une courte explication, veuillez cliquer/tapoter le nom de langue dans le liste ci-dessous.

    • Python

      Primary programming language for personal projects from 2021 onward. Significant projects include the WillPress blog engine, a conlang word generator, and a text-based adventure game engine. Python is the primary language of my 2021 webdev boot camp and the IT Automation with Python professional certificate course through Google underway as of early 2024 as well as an upcoming graduate-level specialization course in data structures offered by the University of Colorado Boulder.

    • C

      Primary programming language from 2008 to 2011. Significant projects included a 3D renderer in OpenGL, system-level utilities including a command shell, an archiver, and an implementation of malloc(3), and projects making use of multithreading and Unix sockets.

    • HTML/CSS

      Familiar with HTML/CSS from the era of CSS1 and HTML4 onward. Significant active projects include this website, but I have used HTML in many contexts over the years.

    • Bash

      Secondary programming language from 2008 onward. No significant projects, but I have written many scenario-specific scripts over the years using Bash, awk, grep, sed, and other Unix command line tools.

    • JavaScript

      Secondary programming language intermittently from 2011 onward. Some familiarity with the React framework and TypeScript language additions. Sole major project is the WillRead frontend for WillPress.

    • PowerShell

      Secondary programming language from 2018 onward. No significant projects, but I use some small PS scripts for my job to streamline my workflow.
